      Sync

      PC·휴대폰 동기화

      지금은 이 브라우저에만 저장됩니다. 나중에 휴대폰과 PC가 같은 체크인·할 일·피드백을 쓰려면 이 설정이 필요해서 메인 화면 대신 여기 숨겨둡니다.

      1Supabase에서 설정 SQL 실행
      2URL·키·프로필 입력 후 연결 확인
      3설정 저장 후 다른 기기에 같은 프로필 입력
      Supabase 설정 SQL 보기

      프로필 키는 동기화 비밀번호처럼 보관하고, 각 기기에 같은 값을 입력하세요.

      저장하면 이 홈 화면 앱의 브라우저 저장소와 IndexedDB에 이중 보관합니다.
